## Anti-Repetition Mechanisms

### How We Prevent Duplicates:

1. **Previous Challenge Tracking**

   - System remembers last 50-100 challenges
   - AI prompt explicitly says "avoid these"
   - Database tracks all completed challenges

2. **Semantic Similarity Detection**

   - AI checks if new challenge is too similar
   - Regenerates if similarity score > 80%
   - Ensures meaningful difference

3. **Creative Constraints**

   - Each generation must include NEW element
   - Must use different action verbs
   - Must have different success criteria

4. **Feedback Loop**
   - User ratings guide future generation
   - Skipped challenges = generate different style
   - Completed challenges = more similar tone
